Durante este año he aprendido distintas maneras de crear mi propia página web con html y aquí voy a mostar como se crea una, y que cosas puede tener la página.

Para crear una página hay que seguir unos requisitos básicos como que esté bien formada y que tenga unas normas básicas.

Las páginas web de cualquier sitio, están formadas por html, css y javascript entre otras muchas cosas. De momento en este curso solo he aprendido html, css y xml y esto es lo que voy a demostar en esta página web.

Puede parecer a simple vista que esta página web es muy sencilla, pero lo más complicado es poder y saber poner una barra lateral a la derecha del texto principal, e insertar un cuadro de búsqueda que lleve directamente a google o a otro navegador, y conseguir que al abrir una página desde otro dispositivo distinto al del origen, esta se adapte al nuevo dispositivo desde el cual se abre y los problemas que genera no sean muy evidentes.







Para poder crear una página web atractiva a la vista del lector, hay que añadirle distintos estilos, y saber de antemano de que va a tratar la web.

En mi caso, solo voy a explicar un poco por encima como se crearía una página web, y como se podrían enlazar dos o más páginas web dentro de un mismo documento html.

Esta página es muy escueta, ya que lo único que quería hacer con ella era meterle todo el css posible con algo de html, y meter imágenes que redireccionaran a los usuarios a otros páginas web. Una de ellas la he creado yo, y la otras son google y wikipedia.

Para acceder a la que he creado, hay que hacer click en el primer icono de los tres que aparecen en la parte superior de este segundo bloque. La segunda imagen, lleva directamente a la página inicial de google y la última lleva a wikipedia (WK). Aunque aparezca el simbolo de Firefox, si se selecciona esa opción, se abrirá una nueva pestaña con google.
